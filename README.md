# OpenGL

## Instalar dependencias

```shell
# Ubuntu 20.04 LTS
sudo apt install freeglut3
sudo apt install freeglut3-dev
sudo apt install libglew-dev
```

## Como rodar

``` shell
cd 01-triangle
make && ./main
```
